# Hardware

* 1x Raspberry Pi 3
* 1x CMPS10
* 1x Line Sensor
* 1x LDR
* 1x Dip Switch
* 4x MicroSwitch
* 5x VL53L0X
* 4x MicroMotors L149.4.90
* 1x Servo Motor
* 1x Dual Motor Driver 1A TB6612FNG
* 1x 128X64 OLED Display Module
